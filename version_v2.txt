## 🚀 NOVIDADES v2.0

### 🛡️ Protocolo Silêncio (EMERGÊNCIA)
- Botão 🚨 vermelho para ativação instantânea
- Comandos: "protocolo silêncio", "silêncio total", "emergência silêncio"
- Fecha abas do navegador automaticamente
- Silencia volume do Windows
- Fecha aplicativos de mídia (Spotify, VLC, etc.)
- Minimiza janelas ativas
- Feedback visual com status "🚨 EMERGÊNCIA"

### 📱 Controle de Aplicativos Aprimorado
- **Busca Dinâmica**: Procura em AppData, Program Files, Registro Windows
- **Validação Real**: Confirma se o aplicativo realmente iniciou
- **Fallback Web**: Se não encontrar localmente, abre versão web
- **Logs Detalhados**: Erros específicos no terminal VS Code
- **Discord Otimizado**: Busca em `app-*\Discord.exe` automaticamente
