## 🎮 INTERFACE DE CONFIRMAÇÃO

### Janela de Confirmação
```
🔌 Desligamento do Sistema

Tem certeza que deseja continuar?

[ ❌ Não ]    [ ✅ Sim ]
```

### Fluxo de Confirmação
1. **Usuário solicita**: "Jarvis, desligar"
2. **Jarvis detecta**: Comando de energia
3. **Interface mostra**: Janela de confirmação
4. **Usuário decide**: Sim ou Não
5. **Jarvis executa**: Ação confirmada ou cancelada

### Mensagens de Feedback
```

❌ "🔌 Ação de energia cancelada, senhor."
😴 "😴 Sistema suspenso, senhor."
```
